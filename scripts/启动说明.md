# UAV 飞行控制平台 — 启动指南

## 启动步骤（按顺序执行）

### 1. 启动 Docker 基础设施（一次性）

```powershell
cd C:\UAVAgent\UAVAgent-main
docker compose -f docker-compose.yml up -d
```

| 服务 | 端口 | 管理界面 |
|------|------|----------|
| MySQL | 3307 | Navicat 连 localhost:3307 |
| Redis | 6379 | - |
| RabbitMQ | 5672 | http://localhost:15672 (uav/uav123) |

### 2. 启动 VcXsrv（Windows 图形服务）

- 开始菜单 → XLaunch → Multiple windows → Display 0 → Start no client → **勾选 Disable access control** → Finish
- 确认右下角托盘有 X 图标

### 3. 启动 PX4 仿真（WSL2 终端 1）

```bash
cd ~/px4v1.14.3
make px4_sitl gazebo-classic
```

### 4. 启动 Bridge 服务（WSL2 终端 2）

```bash
python3 -m uvicorn bridge:app --host 0.0.0.0 --port 8000
```

### 5. 配置端口转发（管理员 PowerShell）

```powershell
cd C:\UAVAgent\UAVAgent-main\scripts
.\setup-portproxy.ps1
```

### 6. 启动后端（IDEA）

运行 `UavAgentMainApplication`

### 7. 启动前端

双击 `C:\UAVAgent\UAVAgent-main\scripts\start-frontend.bat`

或命令行：

```powershell
cd C:\UAVAgent\UAVAgent-main\uav-web
npm run dev
```

浏览器访问: **http://localhost:5173**
